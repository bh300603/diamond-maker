Diamond Maker

Description:

This is a Javascript based generative web application. Use the sliders on the right to manipulate the diamonds.

Variables:

  • points: This controls the number of points in each diamond.
  • stroke_weight: This controls the weight of the stroke in each diamond.
  • red: This controls the amount of red in each diamond.
    green: This controls the amount of green in each diamond.
    blue: This controls the amount of blue in each diamond.
    opacity: This controls the translucency of each diamond.
    stroke_color: This controls the color of the stroke.
    canvas_color: This controls the color of the background.
